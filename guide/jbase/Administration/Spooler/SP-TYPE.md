# SP-TYPE

Description  The command assigns or reassigns a formtype to a formqueue. It takes the general form:  SP-TYPE formqueue formtype where:  formqueue is the name of the formqueue for form type assignment. formtype is the name of the formtype to be used for formqueue configuration. To revert a formqueue to the default form type : SP-TYPE formqueue - Note
