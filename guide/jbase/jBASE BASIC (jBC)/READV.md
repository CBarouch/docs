# READV

Description The READV statement allows a program to read a specific field from a record in a previously opened file into a variable. It takes the general form: READV variable1 FROM { variable2,} expression1, expression2 {SETTING setvar} {ON ERROR statements} {THEN|ELSE statements} Where: variable1Â is the identifier into which the record will be read
