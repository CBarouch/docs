# BYTELEN

Description The BYTELEN function returns the length of the expression as the number of bytes rather than the number of characters. It is used as: BYTELEN(expression) Where expression can return a result of any type. The BYTELEN function will then return the byte count of the expression. Note: The BYTELEN function will always return the actual byte c
