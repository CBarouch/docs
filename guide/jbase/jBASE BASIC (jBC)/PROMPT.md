# PROMPT

Description Used to change the PROMPT character used by terminal input commands. It takes the general from:Â  PROMPT expression Where expression can evaluate to any printable string. The entire string will be used as the prompt. The default prompt character is the question mark ? character. An example of use is as: PROMPT "Next answer: " INPUT Answer
